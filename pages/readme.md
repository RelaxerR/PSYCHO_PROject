- **`pages/`** — отдельные логические страницы или их основное содержимое.  
  Каждый файл представляет собой контент одной страницы (например, `about.php`, `contact.php`, `profile.php`),  
  который подключается через `index.php` или простой роутинг. Не содержит полной HTML-разметки — только тело страницы.
