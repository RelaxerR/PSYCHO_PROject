- **`src/`** — централизованное хранилище всех клиентских (фронтенд) ресурсов: стилей, скриптов, изображений и шрифтов.  
  - **`assets/css/`** — CSS-файлы (например, `style.css`, `reset.css`, `components/` для модульных стилей).  
  - **`assets/js/`** — JavaScript-файлы (например, `main.js`, `utils.js`, `modules/` для отдельных функций или компонентов).  
  - **`assets/img/`** — изображения сайта (логотипы, иконки, баннеры и т.д.; можно организовать подпапки: `icons/`, `banners/`, `avatars/`).  
  - **`assets/fonts/`** — локально подключаемые веб-шрифты (`.woff`, `.woff2` и др.).
