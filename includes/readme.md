- **`includes/`** — переиспользуемые PHP-фрагменты и служебные файлы, подключаемые в другие скрипты.  
  Сюда относятся:  
  - `header.php`, `footer.php`, `nav.php` — общие части макета;  
  - `config.php` — настройки подключения к БД, константы, пути;  
  - `functions.php` — вспомогательные функции (валидация, работа с файлами и т.п.).
